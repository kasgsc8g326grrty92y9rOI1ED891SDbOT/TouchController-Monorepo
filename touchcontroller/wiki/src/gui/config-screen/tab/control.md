# 标签页：控制

- 此标签页属于“通用”分类
- 此标签页中的所有选项均为滑杆

## 设置项

### 视角移动灵敏度

- 游戏内描述：设置横向滑动整个屏幕时视野旋转的角度。
- 默认值：495（单位：度）
- 控制手指划过一个屏幕宽度时视野旋转的角度。

### 长按检测阈值

- 游戏内描述：设置判断屏幕点击是长按还是移动所需要的最小移动距离。
- 默认值：2（单位：屏幕宽度的百分比）
- 如果手指按下后移动的距离超过了这个数值，则判定为滑动屏幕。

### 长按检测时间

- 游戏内描述：设置判断屏幕点击是否为长按所需要的最小按下时间。
- 默认值：5（单位：客户端刻）
- 如果手指按住不动的时间超过了这个数值则判定为长按。
